JS STATEMENTS

  • In programming language, instructions give to computer are called statements.
  • Javascript program is a list of programming statements.
  • Javascript statements composed of: Values, Operators, Keywords and Comments.
Semicolons (;)
Example:
a=5;b=3;c=2;
i Semicolons are used to seperate javascript statements.
White Space in JS ( )
Example:
var person = "Hola";
var person="Hola";
Two lines above are same.
i White Spaces are ignored in Javascript. Good practice is to have space around operators like +,-,=,*,etc.
Code blocks
Example:
function Example_func(){
    var person = prompt("Enter");
    console.log(person);
}
i Code blocks are used to define statements that are to be executed together.

JS KEYWORDS

Keyword Description
break Terminates a switch or a loop.
continue Jumps out of a loop and starts at the loop.
debugger Stops the execution of JavaScript, and calls(if available) the debugging function.
do ...while Executes a block of statements, and repeats the block, while a condition is true.
for Marks a block of statements to be executed, as long as condition is true.
function Declares a function
if ... else Marks a block of statements to be executed, depending on a condition.
return Exits a function.
switch Marks a block of statements to be executed, depending on different cases.
try ... catch Implements error handling to a block of statements.
var Declares a variable.

JS STRINGS

Strings in JS are collection of characters.

We can initialize strings as objects but it slows down the website.

Examples:

Adding Strings
var x = "Hello" + 2 + 3;
>>>   Hello23

var y = 3 + 5 + "Car";
>>>   8Car
i Javascript evaluates expression from left so in second example it will add numbers first then add it to string.
String Can Be Objects
var x = "String";
var y = new String("String");
x==y;
>>>  true
x===y;
>>>  false
i First is true because there value are same. But in second one they're not of same type.
Comparing Objects
var x = new String("String");
var y = new String("String");
x==y;
>>>  false
x===y;
>>>  false
i Both are returning false because objects can't be compared in Javascript.

Some methods for strings:

Length Method
var x = "Hello";
x.length;
>>>   5
i Length method returns the length of string.
Finding A String
var x = "String";
x.indexOf("t");
>>>  1
x.lastIndexOf("i",4);
>>>  1
i It will return the first occurence of the string to be found in given string. Second argument specifies leftmost position from where it starts and proceed towards ending. returns -1 if string is not found.
Finding a string from the last
var x = "Strinig";
x.lastIndexOf("i");
>>>  5
x.lastIndexOf("i",4);
>>>  3
i It searches from the right and returns first occurence from the "right". Second argument specifies rightmost position from where it starts and proceed towards beginning. returns -1 if the string is not found.
Searching a string
var x = "Strinig";
x.search("i");
>>>  3
i Same as indexOf but cannot take second argument. It can take regular expressions. indexOf cannot take regular expressions.
Slicing a String
var x = "Strinig";
x.slice(3,6);
>>>  ini
i Takes out the specified part of string. First argument specifies start and second argument stop-1 .
Slicing a String using slice
var x = "Strinig";
x.slice(3,6);
>>>  ini
x.slice(-4,-2);
>>>  in
i Takes out the specified part of string. First argument specifies start and second argument stop-1 . If you omit the second parameter the it will slice out the rest string.
Slicing a String using Substring
var x = "Strinig";
x.substring(3,6);
>>>  ini
i Same as slice. but it cannot negative indexes. If you omit second it will slice out rest string.
Slicing a String using Substr
var x = "Strinig";
x.substr(3,4);
>>>  inig
i It also slices out the string. First argument specifies starting position and second argument specifies required length. omiting second argument will return rest string.
Replacing String
var x = "Strinig";
x.substr("i","Ho");
>>>  StrHonig
i It doesn't changes the string. It return new string. Replaces the first occurence of given string. Regular expression are not written in quotes.
Changing Case
var x = "Strinig";
x.toLowerCase();
>>>  strinig
x.toUpperCase();
>>>  STRINIG
i toLowerCase changes case to lowercase. toUpperCase changes case to uppercase.
Concatinating Strings
var x = "Strinig";
x.concat(" ","broom","hola");
>>>  Strinig broomhola
i you can put as many arguments as you want they will all add up to make one string.
Trimming Strings
var x = "   Strinig   ";
x.trim()
>>>  Strinig
i It removes all white spaces before and after the string.
String Padding
var x = "   Strinig   ";
x.trim()
>>>  Strinig
i It removes all white spaces before and after the string.
Extracting String Characters
var x = "Strinig";
x.charAt(0) >>>  S
i Extracts the character at specified position.
Extracting String Character using []
var x = "Strinig";
x[0];
>>>  S
i same as charAt(). If no character found it returns undefined while str charAt() returns empty string. Doesn't give error when we try to change value but doesn't change the value either. as strings are immutable.
Converting String to an Array
var x = "a,b,c,d,e,f,g";
x.split(",");
>>>  ["a","b","c","d","e","f","g"]
i separates the string from the character provided and stores them in a list.
